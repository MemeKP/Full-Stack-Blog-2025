### Overview

PW Blog is a full-stack blogging platform developed to provide a central space for who wants to **share their personal stories, experiences, and ideas** with a wider community. Built on the **MERN stack**, it offers robust functionality for post creation, reading, and engaging through comments and likes, all within a responsive environment.

### Key Features

* **Content Management:** Supports post creation, editing, and publishing.
* **User Engagement:** Functionality for comments, likes, and post bookmarks.
* **Secure Authentication:** Integrated user sign-up, login, and role-based access control.
* **Responsive Design:** Dynamic and user-friendly interface built with React, ensuring compatibility across various devices.
* **RESTful API:** Efficient handling of data transactions between the front-end and back-end.

### Tech Stack

| Component | Technology | Description |
| :--- | :--- | :--- |
| **Frontend** | JavaScript (React) | Used for building a component-based, dynamic, and user-friendly interface. |
| **Backend / APIs** | JavaScript (Node.js/Express) | Built to handle business logic and manage data requests. |
| **Database** | MongoDB | Managed persistent data storage for posts, users and comments data. |
| **Authentication** | Firebase | Integrated for secure user authentication and role-based access control. |
| **Deployment** | Vercel | Platform deployed using Vercel. |
